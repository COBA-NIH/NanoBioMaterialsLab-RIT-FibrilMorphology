# Customizing a fiber segmentation model with ilastik and CellProfiler

This folder contains a cellprofiler pipeline, ilastik project file, and minimal training sets for segmenting fluorescent fibers. More details can be found in [Customizing a Model for Fiber Segmentation](https://carpenter-singh-lab.broadinstitute.org/blog/customizing-model-fiber-segmentation-part-i-investigating-possible-methods).
